Access Control Reports


Includes [All access control events], [Access control exception events] and [Personnel access levels] reports. You can select Export all and Export after query.

The user can generate statistics of relevant device data as per reports, including card verification information, door operation information, and normal card punching information, etc.

About the Normal event and abnormal event, please refer to Real-time Monitoring for details.

Verify mode: Only Card, only password, password plus card and others.

Note: Only event records generated when the user uses emergency password to open doors will include [Only password] verification mode.
