Network Video Recorder


The system provide video linkage function, to manage the network video recorder, view the real-time video, and query the video record, popup the real-time video when the linkage events happened.

The system provide video linkage function, to manage the network video recorder, view the real-time video, and query the video record, popup the real-time video when the linkage events happened.

When using the video linkage function, the user need to add a network video recorder firstly, for detailed operation, please refer to Add device. After adding, the user need to set video linkage in linkage settings (set the network video recorder and bound channel), then the user can view the real-time monitor video in the system. The detailed setting, please refer to Linkage Setting.

Video Preview: Enter [Video]-[Video Preview], open the video preview interface, select the preview channel to view the remote preview image.

Double click the image, enter the full screen mode.

Double click the image, or click [Windows] icon on your keypad, or press Ctrl+ESC to exit the full screen mode.

Linkage Video Records: Enter [Video]-[Linkage Video Records], to query or playback the linkage video records.

Select the corresponding control panel, door or network video recorder, input the start and end time, click [Search the Video] button. If the record matches the condition, the record number and name will display on the list, and the play and stop operation is permitted.

Click [Play] button, the video will playback on the right interface, click [Stop] to stop it.

The full screen operation is as the description above.

Alarm Preview: After setting the linkage event, if the real-time linkage event happens, the preview interface will pop up in the real-time monitor interface.
